## How to Build and Run

### Prerequisites

- Make (Optional)
- C++11
- Windows

### Build and Run

#### Option 1 (With Make)

```cmd
:: 1. Clone GitHub repository
git clone https://github.com/anthonyandrei/CSOPESY-Marquee_Console
cd CSOPESY-Marquee_Console

:: 2. Build and run
make build run
```

#### Option 2 (Without Make)

```cmd
:: 1. Clone GitHub repository
git clone https://github.com/anthonyandrei/CSOPESY-Marquee_Console
cd CSOPESY-Marquee_Console

:: 2. Build and run
g++ -std=c++11 -pthread -o main main.cpp
./main
```
